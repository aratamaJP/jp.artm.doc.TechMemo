Vueの使用方法としては

CDNを使う方法 vue-cliを使う方法がある 他にもnpmでインストールする方法や、 Webプラットフォームを使う方法もある

CDNを使う場合

<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

vue-cliを使う場合

vue-cliのインストール

node.jsのnpmコマンドでインストールを行う
*node.js 6.x以降が必要 npm install -g @vue/cli npm install -g @vue/cli-init * 下記のコマンドで実施すると古いバージョンばインストールされるので注意 # npm install -g vue-cli

プロジェクトの作成

vue init webpack-simple [ Project Name ]

webpack： webpack、 lint、 ユニット テスト などを 組み込ん だ プロジェクト webpack-simple： webpack ベース の シンプル な プロジェクト

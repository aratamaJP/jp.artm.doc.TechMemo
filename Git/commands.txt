■   新しいブランチを作成する

    git checkout master
    git branch [BRANCH_NAME]
    git checkout [ BRANCH_NAME ]


■  リモートに存在しない、ローカル作業ブランチを削除する（PowerShell)

    git pull --prune
    git branch --merged ` | Select-String -NotMatch -Pattern "(\*|main|master|staging)" ` | %{ git branch -d $_.ToString().Trim() }


■ ソースコード追跡を保持したまま、ファイル名を変更

    git mv [現在のファイル名] [新しいファイル名]

■ ソースコード追跡を保持したまま、フォルダを移動

    git mv [移動前フォルダパス]/ [移動後フォルダパス]/

■ リベースの実行

    git checkout [ target-branch ]
    git pull

    [default]
    git pull --rebase origin master
    git pull --rebase origin main

    [flexceed]
    git pull --rebase origin staging

    git rebase --continue

    git push --force

//--------------------------------------------- // Hosting 1. Firebaseにブラウザでログインし、 サイドバーの「Hosting」を選択し「始める」をクリックする。 プロジェクトを作成する。 2. Hostingの機能を初期設定する firebase init [Hosting] を選択する [Use an existing project] 使いたいプロジェクトを選択して Enter 静的ファイルを設置するフォルダを指定する。 デフォルトのままなら「Public」 Reactを使うなら「build」と書いてEnter Vueを使うなら「dist」 single-page app なら 「y」を選択 (Reactの場合はy) * ここまでで .firebasercとfirebasejsonのファイルが作成される 3. Reactのプロジェクトをビルドする npm run build * コンパイルされる public, src => build

Firebase上にデプロイする場合

firebase deploy
* プロジェクトのルートディレクトリで実施する

ローカルでホスティングのテストをする場合

firebase serve --only hosting

* デプロイ先のプロジェクトを変更するとき firebase use [ Project ID ]
